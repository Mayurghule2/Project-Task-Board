# Project Task Board

A full-stack task management application built as an assignment project using ASP.NET Core Web API, SQLite, React, Tailwind CSS, Framer Motion, and Lucide Icons.

## Features

### Backend
- Project CRUD operations
- Task CRUD operations
- Comments module
- Filtering, sorting and pagination
- Dashboard analytics API
- Global exception handling
- SQLite database
- Layered architecture (Controllers, Services, DTOs)

### Frontend
- Premium responsive dashboard UI
- Project management page
- Task board with filters
- Task detail page with comments
- Framer Motion animations
- Tailwind CSS styling
- Lucide icons

---

## Tech Stack

### Backend
- ASP.NET Core 8 Web API

- SQLite

### Frontend
- React.js
- Vite
